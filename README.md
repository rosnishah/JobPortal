# WorkOwl - Job Portal Web Application

WorkOwl is a fully dynamic web application developed using C# and the ASP.NET framework. The platform is engineered to bridge the gap between corporate recruiters and job seekers by providing an intuitive, structured environment to post vacancies, manage profiles, and track career applications seamlessly.

## 🚀 Key Features

* **Recruiter Dashboard:** Dedicated secure interface for corporate entities to publish detailed job descriptions, manage active listings, and filter candidate pools.
* **Job Seeker Module:** Comprehensive registration and authentication flow allowing users to set up professional profiles, upload resumes, and track their application lifecycle.
* **Advanced Vacancy Search:** Implements multi-conditional server-side filtering to search for open roles based on specific skill sets, geographical location, and job types.
* **Modern UI Architecture:** Structured with intuitive navigational components including dedicated About Us, Services, and interactive Contact portals to mimic real-world enterprise platforms.

## 🛠️ Tech Stack & Requirements

* **Backend Engine:** C# (C-Sharp)
* **Web Framework:** ASP.NET (MVC / Web Forms)
* **Database Management:** Microsoft SQL Server / MySQL (Structured Relational Schema)
* **Frontend Blueprint:** HTML5, CSS3, JavaScript, and Figma prototypes
* **IDE Support:** Visual Studio 2022 / 2019

## 📂 Project Architecture

The repository follows a clean separation of concerns:
* `/Controllers` or `/CodeBehind` - Handles core business logic and request routing.
* `/Views` or `/Pages` - Holds the structural layout components (HTML/CSS) for the user interface.
* `/Models` - Maps data objects to the underlying SQL relational database tables.
* `/bin` - Contains compiled binaries (`.dll` files) orchestrated by the IIS Express server.
